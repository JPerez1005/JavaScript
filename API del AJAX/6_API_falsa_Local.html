Para Instalar JASON server como API falsa local primero se instala

con el siguiente codigo:




npm install -g json-server




si ya lo teniamos instalado entonces pasamos al siguiente paso

despues ejecutamos el siguiente comando junto con la ruta

del json:


json-server --watch db.json




En este caso el archivo esta dentro de la carpeta json dentro de

dentro de la carpeta API del AJAX



La manera de cambiar de puerto es: json-server -w -p 5000 json/db.json
