Undefined

cuando no definimos una variable, es decir si creamos una varible y no le colocamos un valor es por que está indefinida.


Null

Así como con indefinido, la palabra reservada null se coloca explicitamente para decir que no definiremos esta variable por que la dejaremos vacía, de modo intencinalmente por el programador.


Nan

Not is an number, no es un numero, es precisamente para decirle al programador o al usuario que la operacion que se esta intentando hacer no es logica o alguna de las variable falla debido a un error de asignación de variables.
