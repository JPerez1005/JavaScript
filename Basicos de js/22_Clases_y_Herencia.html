Clases y Herencía

A partir del 2015 cambiaron el estandar en JavaScript y podremos cambiar de objetos
con metodos constructores a clases que JavaScript los compila como prototipos es
decir ya no necesitaremos de funciones constructoras, se hizo para simplificar codificación

Aclarar que esto no hace a JavaScript un lenguaje orientado a objetos basado en clases
Sigue siendo un lenguaje orientado a onjetos basado en prototipos
